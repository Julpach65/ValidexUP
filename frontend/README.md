# Validex UP - Frontend

Este es el espacio para el desarrollo de la interfaz de usuario.

## Stack Tecnologico
- Next.js 14+ (App Router)
- Tailwind CSS
- shadcn/ui
- react-webcam
- Zustand

## Instrucciones para el Miembro 1:
1. Navega a esta carpeta: `cd frontend`.
2. Inicializa el proyecto con: `npx create-next-app@latest .`.
3. Sigue las configuraciones de diseño detalladas en la carpeta de la raiz del proyecto.
